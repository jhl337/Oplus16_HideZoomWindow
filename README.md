# Oplus16_HideZoomWindow
从截图和录屏中隐藏小窗的Xposed模块(为ColorOS16设计)。

## 支持的设备
- 仅在 一加 Ace6 (ColorOS 16.0.2.302&402&403) 测试通过。
- 大部分 ColorOS 15 设备也可以进行使用，自行测试。
- **不保证任何其它机型或其它版本的可用性。**

## 功能
- 从截图和录屏中隐藏小窗。
- 从截图和录屏中隐藏侧边栏。
- 从截图和录屏中隐藏输入法。
- 从录屏中隐藏截图的预览窗。
- 防止焦点切换到小窗(防止某些软件检测小窗)。
